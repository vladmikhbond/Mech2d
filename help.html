Параметри моделі

W - ширина модельного простору (10 < W < 10000)
H - висота модельного простору (10 < H < 10000)

k - жорсткість матеріалу куль (10 < k < 1000)
w - коеф.втрат енергії при зіткненні куль (0 <= w <= 1) 0 - зіткнення без втрат 
u - коеф.втрат енергії при зіткненні кулі з лінією (0 <= u <= 1) 0 - зіткнення без втрат 
v - коеф.втрат енергії від спротиву повітря (0 <= v <= 1) 0 - спротиву повітря немає
Після змінення параметрів натисніть Enter, щоб зміни почали діяти.

Режими створення сцен

Inf - виводить дані про об'єкт, що під курсором миші
Sto - будує лінійні перешкоди
Osc - будує точкові і лінійні осцилятори
Mon - будує точкові і лінійні однократні-осцилятори
Rec - будує точкові і лінійні приймачі
Del - малює прямокутнік и видаляє все, що в нього потрапило

Параметри майбутнього осцилятора

amp - амплітуда
q   - хвильове число = 2π/λ - частина фази коливання, яка припадає на одиницю довжини. 
vx  - швидкість горизонтального руху (1/20 < vx < 1/2)

Параметри майбутнього приймача

loss - коеффіцієнт поглинання  (0 < loss < 1)

Керування переглядом

Кнопка Run   - зупиняє або продовжує плин часу
Кнопка 3d/2d - перемикад відображення
Клавіша s    - один крок у часі
Слайдер масштабу - змінює масштаб відображення хвиль
